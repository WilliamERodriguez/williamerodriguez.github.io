Projects

The projects listed here focus on applying foundational Linux and IT concepts in controlled, repeatable environments. Each project emphasizes system understanding, documentation, and practical problem-solving rather than production deployment.

Linux Fundamentals Lab

A structured learning environment used to practice and document core Linux system concepts through hands-on experimentation.

This project was created to build familiarity with Linux systems in an isolated environment, allowing experimentation without risk to a primary system. Work is performed directly from the command line to reinforce understanding of system behavior.

Focus Areas

  • File system navigation and layout
  • User accounts and file permissions
  • Basic process monitoring and management

Approach

A Linux virtual machine is used as a sandbox environment. Tasks are performed manually to observe how the system responds to commands and configuration changes. Notes are recorded to document observations and repeatable steps.

Tools

  • Linux (Fedora or Debian-based distributions)
  • Bash shell
  • Virtualization software

This project is ongoing and serves as a foundation for continued system-level learning.
