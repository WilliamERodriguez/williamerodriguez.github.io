Linux Fundamentals Lab

A structured virtual environment used to practice and document core Linux system concepts.

Context

This project was created to build hands-on familiarity with Linux systems in an isolated environment, allowing experimentation without risk to a primary system.

Scope

  • Basic file system navigation
  • User accounts and permissions
  • Process monitoring and management

Approach

A Linux virtual machine was configured and used as a sandbox environment. Tasks were performed manually to reinforce understanding of system behavior and command-line tools. Notes were recorded to document observations and repeatable steps.

Tools & Technologies

  • Linux (Fedora / Debian-based distribution)
  • Bash Shell
  • Virtualization Software

Outcome

This project improved familiarity with Linux system structure and increased confidence working with the command line. It also established a repeatable process for experimenting with system-level concepts safely.
